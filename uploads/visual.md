# Sudoku Dojo — Visual Concept Test

## Goal

Test 3 visual directions before building anything else. Build 2 screens in all 3 variants. No flows, no real interactions — visuals only. The user picks one direction, then the full build begins.

---

## Screens to Build

### Screen 1 — Game Screen
The grid is the core of the product. If the visual direction works here, it works everywhere.

Must show:
- 9x9 Sudoku grid with some numbers pre-filled and some user-entered
- One cell selected, related cells (same row/col/box) subtly highlighted
- Left sidebar: timer, difficulty rating badge
- Right sidebar: number pad (1–9), erase, hint, undo, notes toggle

### Screen 2 — Rank Selection (Onboarding Step 2)
The most branded screen. First real impression of the identity.

Must show:
- Heading: "Choose Your Path" with Japanese subtitle
- Four rank cards: 白帯 Shiro-obi, 緑帯 Midori-obi, 黒帯 Kuro-obi, 師匠 Shishou
- One card in a hover/selected state

---

## The 3 Variants

Build both screens in each variant. Show all variants so they can be compared.

### Variant A — Zen Ink
The feeling of a Japanese tea ceremony. Washi paper, ink brushstrokes, quiet ritual. Like writing numbers with a fine pen on handmade paper. Calm, analog, slightly imperfect in a beautiful way.

### Variant B — Liquid Glass Dojo
Modern and slightly futuristic. Glass morphism as the primary material — frosted panels, depth, blur. Underneath the glass: faint Japanese motifs. Like looking at a zen garden through floor-to-ceiling glass in a modern Tokyo building. Premium, clean, Apple-adjacent.

### Variant C — Pixel Dojo
Nostalgic and playful. Pixel art gamification inspired by retro Japanese games — early Nintendo, Famicom Japan aesthetic. Numbers have a handcrafted pixel quality. Cozy like Stardew Valley but set in a Japanese dojo.
